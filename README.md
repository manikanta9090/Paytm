# Paytm Wallet Clone

A modern fintech wallet application built with Next.js, Prisma and PostgreSQL.

## Features

- Wallet balance management
- Add money via bank
- Send money (P2P transfers)
- Transaction history
- Live market dashboard (USD, EUR, BTC, NIFTY, SENSEX)
- Modern Paytm-style UI
